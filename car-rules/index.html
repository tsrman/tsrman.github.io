Авто-Правила — инструкция пользователя

Простое руководство по приложению «Авто-Правила» для головного устройства Seres M5 с прошивкой 2510.

Поддержать разработку: https://pay.cloudtips.ru/p/6e36be30

Что это

Приложение позволяет создавать автоматические правила для автомобиля по принципу:

КОГДА что-то произошло → И ЕСЛИ выполняются условия → ТО выполнить действия.

Пример: «Когда водитель сел на сиденье, и если на улице ниже −5 °C, то включить подогрев руля и сиденья на 15 минут».

Приложение работает в фоне (запускается само после включения головного устройства).

Первый запуск

  1. Установите APK и откройте приложение.
  2. Разрешите доступ к геолокации — он нужен для геозон (см. ниже). Если геозоны не нужны, можно отказать.
  3. При старте раздаётся короткая мелодия — значит, приложение подключилось к автомобилю.
  4. Уже созданы 4 готовых правила: подогрев руля и сидений водителя/пассажира при холодной погоде. Их можно отключить, изменить или удалить.

Главный экран

  • Правила — список всех правил. У каждого: переключатель вкл/выкл, кнопки «Запустить» (проверить вручную), «Изменить», «Удалить».
  • Состояние автомобиля — живые значения параметров авто (температура, скорость, ремни, двери и т.д.).
  • Журнал срабатываний — история: какое правило когда сработало и что сделало, включая ошибки выполнения действий.

Как создать правило

  1. На главном экране нажмите «Правило».
  2. Введите название (например, «Прогрев зимой»).
  3. Заполните секцию «Когда (триггер)»: - Параметр / событие — что отслеживать (например, «Датчик сиденья водителя»). Под полем видно текущее живое значение этого параметра в авто — удобно сверить коды. - Переход — тип события: «Стало равно», «Изменилось», «Возросло», «Уменьшилось». - Значение — для «Стало равно»: до какого значения (например, «1 — Занято»). - Кнопка «Триггер» добавляет ещё один триггер — правило сработает при любом из них (ИЛИ).
  4. При желании добавьте условия («И если»): параметр, знак сравнения, значение. По умолчанию требуются все условия (И); переключателем можно выбрать «Любое из условий» (ИЛИ).
  5. Добавьте действия («То выполнить»): что включить и на какое значение (например, «Подогрев руля» = «3 уровень»). - Поле «Минут» — через сколько минут автоматически выключить обратно (0 — не выключать). Таймеры автовыключения переживают перезагрузку и сон. - Галочка «Задержка перед выполнением» — выполнить действие через N секунд после срабатывания правила. - Кнопка «тест» у действия — немедленно выполнить его один раз и проверить, что значение верное.
  6. Нажмите «Сохранить».

Полезные опции

«Срабатывать при старте»

Галочка в секции триггера (для перехода «Стало равно»). Без неё правило реагирует только на момент изменения параметра. С галочкой оно дополнительно сработает при запуске приложения, если значение уже нужное — например, авто включилось вместе с головным устройством, и перехода «выкл → вкл» приложение не видело. Защита от повторов: правило не сработает дважды за одну поездку (перевзводится, когда питание отключается, либо по таймауту 4 часа).

Геозоны

Галочка «Также требовать нахождение в зоне» открывает блок геозоны: задайте радиус в метрах, доедьте до нужного места и нажмите «Сохранить текущие координаты» — точка запомнится.

  • Триггер оставлен пустым → правило срабатывает при въезде в зону (например, «приехал домой — открыть лючок зарядки»).
  • Триггер задан (например, «Передача стала P») → геозона работает как дополнительное условие: правило сработает только внутри зоны.

Координаты берутся с GPS головного устройства, не из шины авто.

Особые действия (не из шины авто)

  • Запустить приложение — открыть любое установленное приложение.
  • Громкость медиа — установить громкость в процентах (шаг 10%).
  • Управление Яндекс Музыкой — играть / пауза / следующий / предыдущий трек.

Как это работает важно знать

  • Правило срабатывает один раз на событие: пока значение держится, повторов нет.
  • Первое значение после запуска считается «базовой линией» и не вызывает срабатывание.
  • Каждое срабатывание сопровождается короткой мелодией и записью в журнале.
  • Коды значений перечислений (уровни подогрева, режимы) подобраны по реальным дампам, но могут отличаться в другой комплектации — проверяйте кнопкой «тест» и живым значением в редакторе.

Что можно автоматизировать (примеры)

  • Подогрев/вентиляция сидений и руля по температуре и посадке.
  • Блокировка дверей при начале движения.
  • Открытие лючка зарядки по приезде домой (геозона).
  • Режим вождения «Снег» при минусе за бортом.
  • Запуск музыки или навигатора при включении авто.
  • Громкость по расписанию событий (например, тише ночью — через условия).

Полный список доступных параметров — в docs/PARAMETERS_RU.md.
